Collage

Steps to Create a collage in gimp

Step 1: Create a new file. Set the width to 600px and height to 400px.

Step 2: Open the other 3 image files.

Step 3: Go to “ghandruk-village-trek.jpg” and copy the file and paste it in the new file. Add a new layer after adding each image.

Step 4: Select Scale tool and click on “ghandruk-village-trek.jpg” image. Then unlink the width and height (the height will be set to 91px if not unlinked first) and set the width to 300px and height to 90px and click on “Scale”.

Step 5: Adjust the image to resemble the sample image as much as possible.

Step 6: Go to the female yoga image and select Ellipse select tool. Set the feather radius accordingly (I have set to 40px), and select the image to resemble the sample image as much as possible.

Step 7: Copy the selected part and paste it in the file and add a new layer. Adjust the image to resemble the sample image as much as possible and set the opacity to 60%.

Step 8: Select Text tool and type in your MIT ID (MIT221996). Change the font-style to Goudy Stout, font-size to 42px, font-color to white and make it bold.

Step 9: Adjust the position of the text to resemble the sample text as much as possible.

Step 10: Go to File and select “Export as” to export the image
